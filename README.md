# Node.js Backend Journey (12 Weeks)

This repository documents my 12-week journey to learn backend development with Node.js.  
Each week I’ll build a new project and track what I learn.

---

## Weekly Progress

| Week | Focus | Project(s) | Status |
|------|--------|-------------|--------|
| 1–2 | Node.js and Express basics | Hello Node, CRUD API | Done |
| 3–4 | Database and Authentication | Task Manager API | In Progress |
| 5–6 | File Uploads and Validation | Blog API | Planned |
| 7–8 | Real-time and Payments | E-commerce, Chat App | Planned |
| 9–10 | Microservices and GraphQL | DevShop | Planned |
| 11–12 | Production and CI/CD | SaaS Backend | Planned |

---

## Goals

- Learn Node.js and Express fundamentals  
- Build REST APIs with authentication and databases  
- Understand caching, testing, and deployment  
- Create production-ready backend projects  
- Document weekly progress

---

## Tech Stack

- Node.js  
- Express.js  
- MongoDB (Mongoose)  
- JWT Authentication  
- Redis  
- Docker  
- Jest
